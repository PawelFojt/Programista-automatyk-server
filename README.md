To repozytorium zawiera kod z backendu do projektu bloga o automatyce.
Na chwilę obecną do poprawnego działania musisz uruchomić frontend z repozytorium Programista-automatyk.
Aby uruchomić backend wpisz w konsoli npm start.
Jeśli wszystko poszło poprawnie zobaczysz w konsoli "połączono z bazą danych", "backend uruchomiony"
